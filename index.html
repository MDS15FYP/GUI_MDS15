About Our Project



Our project is "Drone-based Emotion Recognition System for Real-time Surveillance" which tackles crucial research problems in modern surveillance.

The project aims to create a real-time face recognition model utilising drones, to enhance security with its tracking capabilities, while also incorporating emotional analysis to identify suspicious activities.
